<section class="section">
    <h2>Selected Artifacts</h2>

    <div class="resume-entry">
        <h3>CYSE 301 – Sword vs. Shield Project</h3>
        <p class="resume-meta">Cybersecurity Techniques and Operations</p>

        <p>
            This project explored both offensive and defensive cybersecurity
            techniques within a controlled laboratory environment. Using tools such as
            Nmap and Wireshark, I performed network reconnaissance, analyzed packet
            traffic, and identified active hosts, services, and operating systems.
        </p>

        <p>
            In the defensive portion of the project, I configured firewall rules in
            pfSense to restrict network traffic and evaluate how defensive controls
            reduce system visibility and limit attack surfaces.
        </p>

        <a class="button" href="{{ '/assets/documents/writing-research/technical-systems-security/CYSE301_Assignment3_SwordVsShield_JamesDavis.pdf' | relative_url }}">View Artifact</a>
    </div>

    <div class="resume-entry">
        <h3>CYSE 270 – Task Automation</h3>
        <p class="resume-meta">Linux System for Cybersecurity</p>

        <p>
            This assignment focused on automating system administration tasks using
            Linux shell scripting and scheduled jobs. I developed scripts to create
            compressed backups, organize files using timestamps, and move backup
            archives into a designated storage location.
        </p>

        <p>
            I also implemented a cleanup script to remove outdated backups while
            maintaining a defined number of recent files. This work strengthened my
            understanding of Bash scripting, file management, automation, and system
            maintenance.
        </p>

        <a class="button" href="{{ '/assets/documents/writing-research/technical-systems-security/CYSE270_Assignment9_TaskAutomation_JamesDavis.pdf' | relative_url }}">View Artifact</a>
    </div>

    <div class="resume-entry">
        <h3>CS 462 – Microsoft Exchange Server Hack Analysis</h3>
        <p class="resume-meta">Cybersecurity Fundamentals</p>

        <p>
            This project analyzed the 2021 Microsoft Exchange Server hack, a major
            cyberattack involving multiple zero-day vulnerabilities. The analysis
            examined how attackers gained initial access, established persistence through web
            shells, and exploited vulnerable systems at scale.
        </p>

        <p>
            The project also evaluated defensive strategies such as patch management,
            monitoring, incident response, and the use of Microsoft security tools to
            detect and remove malicious artifacts.
        </p>

        <a class="button" href="{{ '/assets/documents/writing-research/technical-systems-security/James Davis - CS462 Blog Term Project - Microsoft Exchange Server Hack.pdf' | relative_url }}">View Artifact</a>
    </div>
